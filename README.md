## Day 1: Getting started

**Step 1 — Confirm your environment is working:**
```
python verify_setup.py
```

**Step 2 — Run your first Mars program:**
```
python example/hello_mars.py
```
(Open the file first and change `"Explorer"` to your name!)

**Step 3 — Connect Roo Code:**
- Click the Roo Code icon in the left sidebar 🦘
- Enter API key: `class2025`
- Enter API Base URL: `https://mars-proxy.creative-ai-builder.workers.dev`
- Switch to **Ask** mode and type: `What is a Sol?`

## Roo Code modes

| Mode | When to use it |
|---|---|
| **Plan** | "I want to build X — help me think through it first" |
| **Build** | "Write the code / make this change for me" |
| **Ask** | "What does this mean? How does this work?" |
| **Fix** | "Something is broken — help me figure out why" |

## Saving your work

Your work auto-saves as you type. To save to GitHub permanently:
1. Click the **Source Control** icon in the left sidebar (branch icon)
2. Write a short note: `added landing sites` or `fixed the map`
3. Click **Commit** → **Sync**

## Coming back after class

1. Go to **github.com** and sign in
2. Click this repository → green **Code** button → **Codespaces**
3. Click your Codespace — opens right where you left off
